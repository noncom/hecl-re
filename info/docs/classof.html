classof
 Hecl Commands 

Name

classof — Returns the name of the internal class of a Hecl Thing.

Synopsis

classof variable

Description

The classof command returns a string containing the Java name of the class that the Hecl variable contains internally.

Example

classof [+ 1 2]

classof "foo"

Produces:

org.hecl.IntThing

org.hecl.StringThing
