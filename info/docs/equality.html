 Hecl Commands 

Name

= != — Integer equality

Synopsis

= num1 num2
!= num1 num2

Description

Tests two numbers for equality, returning 1 if they are, 0 if they aren't equal. In the case of !=, returns 1 if they are not equal, 0 if they are equal.

Example

puts [= 1 1]
puts [= 0 1]
puts [= 00001 1]
puts [!= 1 1]

Produces:

1
0
1
0
