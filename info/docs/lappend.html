lappend
 Hecl Commands 

Name

lappend — Append an element to a list.

Synopsis

lappend listreference element

Description

The lappend takes a reference to a list, and an element to add to that list.

Example

set foo a
lappend $foo "b"
puts $foo
lappend $foo "c d"
puts $foo

Produces:

a b
a b {c d}
